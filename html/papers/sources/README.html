官方资源链接汇总

由于网络限制，直接下载官方 PDF 试卷较为困难。以下汇总了各类公开、可访问的官方来源。

一、官方渠道

1. 北京教育考试院

  • 主页：https://www.bjeea.cn/
  • 中考栏目：每年 6 月中下旬发布当年试题、答案
  • 试题公告：https://www.bjeea.cn/html/zkzz/index.html
  • 说明：官方最权威、最准确的试题来源

2. 北京教育科学研究院

  • 主页：https://www.bjesr.cn/
  • 名师解析、试题分析、命题解读

3. 北京市教育委员会

  • 主页：https://jw.beijing.gov.cn/
  • 中考政策、考试大纲

二、主流教育资源平台

1. 学科网（zxxk.com）

  • 中考真题库：https://www.zxxk.com/
  • 说明：需注册登录/付费，题量最全
  • 涵盖：真题、模拟题、一模二模、名校期中期末

2. 菁优网（jyeoo.com）

  • 主页：https://www.jyeoo.com/
  • 说明：需注册登录/付费
  • 特色：题目详解、类似题推荐、错题本

3. 21世纪教育网

  • 主页：https://www.21cnjy.com/
  • 说明：需注册，部分免费

4. 中考网

  • 北京中考：http://bj.zhongkao.com/
  • 说明：政策解读、复习指南、部分真题

五、如何找到最新试题

每年 6 月 24-26 日（北京中考时段）

  1. 中考结束当天下午，北京教育考试院官网公布试题
  2. 各大教育机构（新东方、学而思等）快速整理版本
  3. 微信公众号推送 PDF

建议关注顺序

  1. 北京教育考试院官网（最准确）
  2. 北京日报客户端（官方媒体）
  3. 主流教辅出版社官网（人教社、北师大、华东师大等）
  4. 教育类公众号（时效性强）

六、免费开源资源

  • 国家中小学智慧教育平台：https://basic.smartedu.cn/
  • 免费提供教材配套资源、习题库、名师课堂
  • 人教版数字教材：https://www.pep.com.cn/

备注

本项目仓库中的试卷内容依据公开题源整理成 Markdown 版本，保留题目原意，格式可能与官方 PDF 略有出入。做题以官方 PDF 为准。如需完整原版 PDF，请到北京教育考试院官网下载或购买官方试题册。
