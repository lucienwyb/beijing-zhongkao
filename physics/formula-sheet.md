# 北京中考物理 · 必背公式速查

## 一、力学公式

| 物理量 | 公式 | 说明 |
|--------|------|------|
| 速度 | $v=\frac{s}{t}$ | $s$：路程，$t$：时间 |
| 密度 | $\rho=\frac{m}{V}$ | $m$：质量，$V$：体积 |
| 重力 | $G=mg$ | $g=9.8$ N/kg（常取 $10$ N/kg） |
| 压强（固体） | $p=\frac{F}{S}$ | $F$：压力，$S$：受力面积 |
| 液体压强 | $p=\rho gh$ | $\rho$：液体密度，$h$：深度 |
| 浮力（阿基米德） | $F_浮=\rho_液 gV_排$ | $\rho_液$：液体密度，$V_排$：排开液体体积 |
| 浮力（称重法） | $F_浮=G-F_示$ | $F_示$：测力计示数 |
| 浮沉条件 | 上浮：$F_浮>G$；悬浮：$F_浮=G$ 且 $V_排=V_物$；漂浮：$F_浮=G$ 且 $V_排<V_物$；下沉：$F_浮<G$ | 区分悬浮/漂浮看排开体积 |
| 功 | $W=Fs$ | $F$：力，$s$：力的方向上移动距离 |
| 功率 | $P=\frac{W}{t}=Fv$ | $P=Fv$ 仅当 $F$ 与 $v$ 同向匀速时 |
| 机械效率 | $\eta=\frac{W_有}{W_总}×100\%$ | $W_有$：有用功，$W_总$：总功 |
| 杠杆平衡 | $F_1l_1=F_2l_2$ | $l$：力臂（支点到力作用线的垂距） |
| 滑轮组（绳子段数 $n$） | $F=\frac{G+G_动}{n}$、$s=nh$ | 不计绳重摩擦；$G_动$：动滑轮重；若忽略 $G_动$ 则 $F=G/n$ |
| 动能 | $E_k=\frac{1}{2}mv^2$（定性） | 质量一定，速度越大动能越大 |
| 重力势能 | $E_p=mgh$（定性） | 质量一定，高度越高重力势能越大 |

## 二、电学公式

| 物理量 | 公式 | 说明 |
|--------|------|------|
| 欧姆定律 | $I=\frac{U}{R}$ | $I$：电流（A），$U$：电压（V），$R$：电阻（Ω） |
| 串联电路 | $I=I_1=I_2$，$U=U_1+U_2$，$R=R_1+R_2$ | 电流相等，电压分压 |
| 并联电路 | $U=U_1=U_2$，$I=I_1+I_2$，$\frac{1}{R}=\frac{1}{R_1}+\frac{1}{R_2}$ | 电压相等，电流分流 |
| 电功 | $W=UIt$ | $t$：时间（s） |
| 电功率（通用） | $P=UI$ | — |
| 电功率（纯电阻） | $P=I^2R=\frac{U^2}{R}$ | 纯电阻电路适用 |
| 焦耳定律 | $Q=I^2Rt$ | $Q$：热量（J） |
| 电能表转数 | $W=\frac{n}{N}$（kWh） | $N$：表盘常数 imp/kWh（或 r/kWh），$n$：实际转数/脉冲数；$W$ 为消耗电能 |

### 串并联电路规律对比

| 物理量 | 串联电路 | 并联电路 |
|--------|---------|---------|
| 电流 | $I=I_1=I_2$（处处相等） | $I=I_1+I_2$（干路=支路和） |
| 电压 | $U=U_1+U_2$（总=分压和） | $U=U_1=U_2$（各支路相等） |
| 电阻 | $R=R_1+R_2$（越串越大） | $\frac{1}{R}=\frac{1}{R_1}+\frac{1}{R_2}$（越并越小） |
| 分压/分流 | $\frac{U_1}{U_2}=\frac{R_1}{R_2}$（串联正比分压） | $\frac{I_1}{I_2}=\frac{R_2}{R_1}$（并联反比分流） |

### 常见电功率记忆
- 普通灯泡：约 20–100 W
- 空调：约 1000–3000 W
- 电冰箱：约 100–200 W
- 电视机：约 100–300 W
- 电热水器：约 1500–3000 W

## 三、热学公式

| 物理量 | 公式 | 说明 |
|--------|------|------|
| 比热容 | $Q_{吸}=cm\Delta t$（升温） | $c$：比热容（J/(kg·°C)） |
| 热量 | $Q_{放}=cm\Delta t$（降温） | $\Delta t$：温度变化量 |
| 热值 | $Q_{放}=mq$（质量）或 $Q_{放}=Vq$（体积） | $q$：热值（J/kg 或 J/m³） |
| 热机效率 | $\eta=\frac{W_有}{Q_总}×100\%$ | — |

### 常见比热容
- 水：$4.2×10^3$ J/(kg·°C) —— 常见物质中比热容较大、最常用的计算对象
- 冰：$2.1×10^3$ J/(kg·°C)
- 酒精：$2.4×10^3$ J/(kg·°C)

## 四、光学公式与规律

### 凸透镜成像（$f$ 为焦距）

| 物距 $u$ | 像距 $v$ | 像的性质 | 应用 |
|-----------|---------|---------|------|
| $u>2f$ | $f<v<2f$ | 倒立缩小实像 | 照相机 |
| $u=2f$ | $v=2f$ | 倒立等大实像 | 测焦距 |
| $f<u<2f$ | $v>2f$ | 倒立放大实像 | 投影仪 |
| $u=f$ | — | 不成像 | 平行光源 |
| $u<f$ | 像距大于物距（不要求计算） | 正立放大虚像 | 放大镜 |

### 光的反射定律
- 反射光线、入射光线、法线在同一平面内
- 反射光线和入射光线分居法线两侧
- 反射角等于入射角
