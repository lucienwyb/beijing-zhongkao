## 缺口原因分析

- **2015-2016 全部**：网页版真题已下架；付费站签名接口未破解
- **2017/2018 非数学**：仅 GitHub 散落 2017 化学英语评析 txt 与 2018 数学 OCR MD，无完整试卷
- **2020 政治**：`zhongkaobj.cn` 未收录，猜测 2020 年北京中考未考或该源未发布
- **2024 生物/历史/地理**：这三科在 `zhongkaobj.cn` 上只有"评析/评论"页，无完整试卷发布页
- **2025 全科**：各站 HTML 快照均为空壳/损坏，无可做题真题；待官方或可靠文字源
- **2026 非数学**：微信公众号主要抢发数学，其余科目多为评析/政策；官方 PDF 考后约 3 个月才发布

## 使用建议

- **备考首选**：`2019-2024` 各科 PDF 已构成 6 年完整题库，主力用它们
- **数学/物理专项**：2019-2024 共 12 份 PDF，是本仓库核心
- **2026 数学**：用整卷图片版查看器做题（官方 PDF 未发布）
- **2025**：暂无可用真题，待可靠来源补充
- **2015-2017**：无完整试卷（2017 仅有化学/英语评析 txt）；2018 数学有 OCR 版 MD（公式排版有损）
