北京中考物理 · 核心考点

按北京中考考纲要求，物理卷覆盖 力学、电学、热学、光学、声学 五大板块。其中力学和电学占比最大（约 70%）。

一、力学（高频重点）

1. 测量与机械运动

  • 长度测量（刻度尺使用、估读）
  • 速度公式 $v=\frac{s}{t}$ 及其变形
  • 匀速直线运动 vs 变速运动
  • 参照物与运动的相对性
  • 近年常考：路程-时间（$s-t$）图像分析（选择题）

2. 质量与密度

  • 质量概念（物质属性，不随形状/状态/位置改变）
  • 天平的使用（调平、称量、读数）
  • 密度公式 $\rho=\frac{m}{V}$
  • 密度的测量实验（排水法测体积、液体密度测量）
  • 近年常考：
  • 密度计算（选填）
  • 测量固体/液体密度的实验题（实验题，5–6 分）

3. 力与运动

  • 力的作用效果（改变形状、改变运动状态）
  • 力的三要素（大小、方向、作用点）—— 力的示意图
  • 重力（$G=mg$，$g=9.8$ N/kg 或 $10$ N/kg）
  • 弹力（弹簧测力计的使用）
  • 摩擦力（静摩擦、动摩擦；增大/减小摩擦的方法）
  • 牛顿第一定律 / 惯性
  • 二力平衡：大小相等、方向相反、同一直线、同一物体
  • 近年常考：
  • 受力分析 + 力的示意图（选填）
  • 摩擦力的判断（选填/实验）
  • 惯性现象解释（简答/选填）

4. 压强

  • 压强定义 $p=\frac{F}{S}$（固体压强）
  • 增大/减小压强的方法
  • 液体压强 $p=\rho gh$（同种液体同一深度压强相等）
  • 连通器原理及应用
  • 大气压强（马德堡半球实验、托里拆利实验）
  • 流体压强与流速的关系（伯努利原理）
  • 近年常考：
  • 压强计算（固体/液体，选填或计算题）
  • 液体压强与深度的关系（实验题）
  • 大气压的应用（吸盘、抽水机等）

5. 浮力

  • 浮力的方向（竖直向上）
  • 阿基米德原理：$F_浮=\rho_液 gV_排$
  • 物体的浮沉条件：$F_浮>G$（上浮）、$F_浮=G$（悬浮/漂浮）、$F_浮<G$（下沉）
  • 轮船（空心法增大浮力）、潜水艇（改变自身重量）
  • 密度计（漂浮时浮力=重力）
  • 近年常考：
  • 浮力计算（选填/计算题，有一定难度）
  • 阿基米德原理实验验证（实验题）
  • 浮沉条件的应用

6. 简单机械

  • 杠杆：五要素（支点、动力、阻力、动力臂、阻力臂）
  • 平衡条件：$F_1l_1=F_2l_2$（$\frac{F_1}{F_2}=\frac{l_2}{l_1}$）
  • 省力杠杆（$l_1>l_2$）、费力杠杆（$l_1<l_2$）、等臂杠杆
  • 滑轮：
  • 定滑轮：不省力，可改变力的方向
  • 动滑轮：省一半力（$F=\frac{G}{2}$），费一倍距离
  • 滑轮组：$F=\frac{G}{n}$（$n$ 为承担重物的绳子段数）
  • 机械效率：
  • 有用功 $W_有=Gh$、总功 $W_总=Fs$（$W_总=W_有+W_额$）
  • $\eta=\frac{W_有}{W_总}×100\%$
  • 滑轮组的机械效率与物重、动滑轮重相关
  • 近年常考：
  • 杠杆平衡条件的探究实验
  • 滑轮组的 $s=nh$ 关系与机械效率计算

7. 功与机械能

  • 做功的两个必要条件（力 + 力的方向上移动距离）
  • 功的计算 $W=Fs$
  • 功率 $P=\frac{W}{t}=Fv$
  • 动能（$m$、$v$）、重力势能（$m$、$h$）、弹性势能
  • 机械能守恒（只受重力/弹力时，动能+势能=定值）
  • 近年常考：
  • 功和功率的计算（计算题）
  • 动能势能转化（选填，单摆、跳高、过山车等情境）
  • 机械能守恒与不守恒的判断

二、电学（高频重点）

1. 电路基础

  • 电路组成（电源、导线、开关、用电器）
  • 串联 vs 并联电路特点
  • 串联：电流处处相等，总电压 = 各用电器电压之和
  • 并联：各支路两端电压相等，总电流 = 各支路电流之和
  • 电流表（串联）、电压表（并联）
  • 滑动变阻器（一上一下、改变接入电阻丝长度）
  • 短路、断路、通路判断
  • 近年常考：电路连接识别与改错（实验题）

2. 欧姆定律

  • $I=\frac{U}{R}$（$U=IR$，$R=\frac{U}{I}$）
  • 电阻的性质（导体本身属性，与 $U$、$I$ 无关）
  • 串并联电路的电阻规律：
  • 串联：$R=R_1+R_2$（越串越大，大于最大的）
  • 并联：$\frac{1}{R}=\frac{1}{R_1}+\frac{1}{R_2}$（越并越小，小于最小的）
  • 近年常考：
  • 动态电路分析（开关开闭/滑片移动引起的 $I$、$U$ 变化）
  • 欧姆定律计算（综合计算题第 20 题）
  • 伏安法测电阻（实验题）

3. 电功率

  • $P=UI$（通用）
  • $P=I^2R$（纯电阻电路）、$P=\frac{U^2}{R}$（纯电阻电路）
  • 额定功率 vs 实际功率
  • 实际电压 = 额定电压 → 实际功率 = 额定功率（正常发光）
  • $U_实 > U_额$ → 灯亮但易烧，$U_实 < U_额$ → 灯暗
  • 焦耳定律：$Q=I^2Rt$（纯电阻：$Q=W=Pt$）
  • 家庭电路：火线零线、保险丝（空气开关）、漏电保护器
  • 安全用电原则
  • 近年常考：
  • 电功率计算（综合计算题第 20 题，最关键的电学计算）
  • 灯泡亮度比较（实际功率越大越亮）
  • 电能表读数与电费计算
  • 动态电路中的功率变化分析

4. 电磁学基础

  • 磁极间相互作用（同名相斥、异名相吸）
  • 磁场与磁感线（从 $N$ 极出发到 $S$ 极）
  • 电流的磁效应（奥斯特实验）
  • 电磁铁（线圈匝数、电流大小影响磁性强弱）
  • 电磁感应（法拉第，发电机原理）
  • 通电导体在磁场中受力（电动机原理）
  • 近年常考：
  • 电动机/发电机原理辨析（选择题）
  • 电磁铁磁性强弱影响因素（实验题）

三、热学

1. 物态变化

  • 六种物态变化：熔化/凝固、汽化/液化、升华/凝华
  • 熔化（固→液，吸热）、凝固（液→固，放热）
  • 汽化（液→气，吸热，有蒸发和沸腾两种方式）
  • 液化（气→液，放热，"白气"不是水蒸气）
  • 升华（固→气，吸热）、凝华（气→固，放热）
  • 晶体（有固定熔点，如冰、海波）vs 非晶体
  • 沸点与气压的关系（气压高，沸点高）
  • 近年常考：物态变化的判断（选择题，结合生活情境）

2. 热量与比热容

  • 温度、热量、内能的区别与联系
  • 改变内能的方式：做功（实质是能的转化）和热传递（实质是能的转移）
  • 比热容概念：$c=\frac{Q}{m\Delta t}$，水的比热容大（$4.2×10^3$ J/(kg·°C)）
  • 热量计算：$Q_{吸}=cm\Delta t$，$Q_{放}=cm\Delta t$
  • 热值：$q$，$Q_{放}=mq$（或 $Vq$）
  • 近年常考：
  • 比热容概念理解（选择题）
  • 热量计算（可能结合实验题或选择题）
  • 水的比热容大的应用（散热、保温）

3. 热机

  • 四冲程汽油机：吸气、压缩（机械能→内能）、做功（内能→机械能）、排气
  • 热机效率：$\eta=\frac{W_有}{Q_总}×100\%$
  • 近年常考：热机四冲程判断（选择题，哪个冲程对外做功？）

四、光学

1. 光的直线传播

  • 条件（同种均匀介质）
  • 现象：影子、日食/月食、小孔成像
  • 光速 $c=3×10^8$ m/s（真空中）

2. 光的反射

  • 反射定律（三线共面、法线居中、两角相等）
  • 镜面反射 vs 漫反射
  • 平面镜成像特点（正立等大虚像，像与物关于镜面对称）
  • 近年常考：光的反射定律探究实验（实验题）

3. 光的折射

  • 折射规律（光从空气斜射入水/玻璃中：折射角 < 入射角）
  • 现象：池水变浅、筷子弯折、海市蜃楼
  • 透镜对光的作用：凸透镜（会聚）、凹透镜（发散）

4. 凸透镜成像

物距 $u$ 像距 $v$ 像的性质 应用
$u>2f$ $f<v<2f$ 倒立缩小实像 照相机
$u=2f$ $v=2f$ 倒立等大实像 测焦距
$f<u<2f$ $v>2f$ 倒立放大实像 投影仪
$u=f$ — 不成像 平行光源
$u<f$ $ v >u$
  • 实验：凸透镜成像规律（必考实验，5–6 分）
  • 近年常考：透镜焦距判断、成像规律应用（照相机/投影仪/放大镜）

五、声学

1. 声音的产生与传播

  • 产生：物体振动
  • 传播：需要介质（固体、液体、气体，真空不能传声）
  • 声速：固体 > 液体 > 气体（$340$ m/s 空气中）
  • 回声计算：$s=\frac{vt}{2}$

2. 声音的特性

  • 音调：频率越高音调越高
  • 响度：振幅越大响度越大（与距离也有关系）
  • 音色：与发声体的材料和结构有关（辨别不同发声体）
  • 近年常考：声音三要素的辨别（选择题，结合古诗词、乐器等情境）

3. 噪声

  • 噪声的定义（物理学 vs 环保角度）
  • 减弱噪声的三条途径：声源处、传播过程中、人耳处

六、能量与能源

  • 能量守恒定律（能量不能凭空产生或消失）
  • 能源分类：一次能源（化石能源、核能、太阳能等）/二次能源（电能）
  • 可再生能源 vs 不可再生能源
  • 核能：核裂变（核电站）vs 核聚变（氢弹）
