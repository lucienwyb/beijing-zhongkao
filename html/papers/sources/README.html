五、如何找到最新试题

每年 6 月 24-26 日（北京中考时段）

  1. 先查询北京教育考试院和北京考试报是否发布试题评析或附件
  2. 再核对主流教辅平台整理版本是否含完整图片、页码与答案
  3. 下载后核验年份、科目、总分、题量和每页完整性，避免使用标题党或模拟卷

建议关注顺序

  1. 北京教育考试院 / 北京考试报（先核验是否真的附有试卷）
  2. 北京日报客户端等官方媒体
  3. 主流教辅或题库平台（必须核对完整图片与答案）
  4. 教育类公众号（只能作为线索，不能仅凭标题确认真题）
